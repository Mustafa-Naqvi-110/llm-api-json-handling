# LLM API JSON Handling

This repository contains beginner-friendly,
well-structured Python examples for working with **Large Language Models (LLMs)** via OpenAI and Anthropic APIs.
It also includes a **CLI tool** for summarizing text files.

---

## Repository Structure

llm-api-json-handling/
├── openai_client/ # OpenAI JSON API examples
│ └── openai_json.py
├── anthropic_client/ # Anthropic JSON API examples
│ └── claude_json.py
├── cli_tools/ # Command-line tools
│ └── summarize.py # CLI text summarizer
├── utils/ # Helper functions
│ └── json_utils.py # Safe JSON parser
├── README.md
├── requirements.txt
└── .gitignore
